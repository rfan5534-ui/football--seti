- index.html : صفحه اصلی
- news.html, teams.html, fixtures.html, about.html, contact.html
- assets/style.css : استایل‌ها
- assets/script.js : اسکریپت ساده
نحوه استفاده:
1) فایل football_site.zip را دانلود و استخراج کنید.
2) پوشه استخراج‌شده را باز کرده و فایل index.html را با مرورگر (گوگل کروم) باز کنید.
3) برای انتشار آنلاین، همین فایل‌ها را روی هاست آپلود کنید یا در GitHub Pages قرار دهید.
